API de Ocorrências Aeronauticas

Com o objetivo de disponibilizar os dados de ocorrências aeronáuticas registradas pelo CENIPA - Centro de Investigação e Prevenção de Acidentes Aeronáuticos - foi criada uma API REST, a partir dos dados disponibilizados pelo Centro no formato .csv, a qual pode ser acessada e consumida por diversas aplicações e/ou serviços.

Consultas Disponíveis

# Endpoint Descrição
Genericas
1 /api/tipos_ocorrencia Obtém todos os tipos de ocorrência registrados.
2 /api/tipos_classificacao Obtém todas as classificacoes para ocorrências registradas.
Ocorrencias
1 /api/ocorrencias Recupera todas as ocorrências registradas.
2 /api/ocorrencias/<:codigo_ocorrencia> Recupera uma ocorrência detalhada selecionada através do código da ocorrência.
3 /api/ocorrencias/<:codigo_ocorrencia>/fatores Recupera os fatores contribuintes para uma ocorrência selecionada.
4 /api/ocorrencias/<:codigo_ocorrencia>/aeronaves Obtém as aeronaves envolvidas em uma ocorrência selecionada.
5 /api/ocorrencias/entre/<:data_inicio>/<:data_fim> Obtém as ocorrências registradas em um intervalo de datas.
6 /api/ocorrencias/uf/<:uf> Recupera as ocorrências registradas em um unico estado.
Aeronaves
1 /api/aeronaves Recupera todas as aeronaves registradas.
2 /api/aeronaves/modelos_mais_ocorrencias Recupera os modelos de aeronave com mais ocorrências registradas.
