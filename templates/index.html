API de Ocorrencias Aeronauticas

Com o objetivo de disponibilizar os dados de ocorrencias aeronauticas registradas pelo CENIPA - Centro de Inverstigaçao e Prevençao de Acidentes Aeronauticos - foi criada uma API REST, a partir dos dados disponibilizados pelo Centro no formato .csv, a qual pode ser acessada e consumida por diversas aplicacoes e/ou serviços.

Consultas Disponiveis

# Endpoint Descricao
Genericas
1 /api/tipos_ocorrencia Obtem todos os tipos de ocorrencia registrados.
2 /api/tipos_classificacao Obtem todas as classificacoes para ocorrencias registradas.
Ocorrencias
1 /api/ocorrencias Recupera todas as ocorrencias registradas.
2 /api/ocorrencias/<:codigo_ocorrencia> Recupera uma ocorrencia detalhada selecionada atraves do codigo da ocorrencia.
3 /api/ocorrencias/<:codigo_ocorrencia>/fatores Recupera os fatores contribuintes para uma ocorrencia selecionada.
4 /api/ocorrencias/<:codigo_ocorrencia>/aeronaves Obtem as aeronaves envolvidas em uma ocorrencia selecionada.
5 /api/ocorrencias/entre/<:data_inicio>/<:data_fim> Obtem as ocorrencias registradas em um intervalo de datas.
6 /api/ocorrencias/uf/<:uf> Recupera as ocorrencias registradas em um unico estado.
Aeronaves
1 /api/aeronaves Recupera todas as aeronaves registradas.
2 /api/aeronaves/modelos_mais_ocorrencias Recupera os modelos de aeronave com mais ocorrencias registradas.
